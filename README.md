Data Analyst based in Gurugram. I work across the full analytics stack — cleaning and transforming data with Python, writing SQL for business questions, building dashboards in Power BI, and running cloud pipelines on AWS.

Most of my projects are end-to-end — raw data in, business insight out. No middle steps skipped.
